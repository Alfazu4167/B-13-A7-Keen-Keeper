

# 🚀 Keen-Keeper # 🏺 Keen-Keeper
**Keep your friendships alive.**

Keen-Keeper is a dedicated relationship-tracking application designed to help you track your friendship bonds.

## 🛠 Technologies Used
*   **React.js** 
*   **Tailwind CSS**
*   **Context API**
*   **JavaScript (ES6+)**

## 🚀 Key Features

1.  **Responsive And User Friendly UI Design**
   This app runs smoothly run in every device, User friendly UI.

2.  **Clean And Reusable Components**
   In This project use context Api to optimized and clean code. 

3.  **Smart Filtering & Live Timeline**
    Easily view text, call, video in timeline section and filter with text, call and video.
